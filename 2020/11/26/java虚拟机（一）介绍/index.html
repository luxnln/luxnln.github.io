简单整理一下java虚拟机的知识。

什么是java虚拟机？

java虚拟机(JVM：Java Virtual Machine)，JVM就是一个虚拟、虚构出来的计算机，它通过在实际的计算机上模拟、仿真各种计算机功能来实现的。这使得Java语言可以通过JVM来屏蔽与具体平台(硬件或操作系统)相关的信息，只需要java编译程序生成可在JVM上运行的目标代码（即字节码），就能在不同不同平台上不加修改的运行。
