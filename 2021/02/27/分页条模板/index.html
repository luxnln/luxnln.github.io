压缩判断条件

然后，我们观察可以知道，我们判断条件为的是确定循环遍历的起始和结尾页数。那么，我们可以使用3原表达式来判断出起始和结尾的页数。

start = page.pages <= 5? 1 : page.pageNum <= 3 ? 1 : page.pageNum > page.pages - 3 ? page.pages - 4 : page.pageNum - 2

end = page.pages <= 5? page.pages : page.pageNum <= 3 ? 5 : page.pageNum > page.pages - 3 ? page.pages: page.pageNum + 2

还做了其它的修改：

  1. 将首页，上一页，下一页，末页的不显示改为禁用，这是为了对称😏；
  2. 跳转请求改用 form 表单来实现，而不是 js 代码。
    1. 一个原因是 thymeleaf 解析 js 好麻烦；
    2. 另一个原因是使用 bootstrap 提供的组件更好对应文字和输入框以及分页条。
